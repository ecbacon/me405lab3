adc::adc ( emstream *  p_serial_port = NULL)

This constructor sets up an A/D converter.

The A/D is made ready so that when a method such as read_once() or read_oversampled is called, correct A/D conversions can be performed. This is accomplished by enabling the ADC, setting the clock prescaler to "divide-by-32", and setting the analog reference voltage to AVCC with an external capacitor to ground. Input should be a serial port. Successful construction is written to the debug.

Parameters:
p_serial_portA pointer to the serial port which writes debugging info.

uint16_t adc::read_average ( uint8_t  channel,
uint8_t  samples 
)

This method takes an average reading on an ADC.

Take an average reading from the adc that is on the channel input as a parameter. The number of samples to be averaged is determined by the parameter "samples". If the "samples" parameter exceeds the maximum permissible number of samples(63), then only 63 samples are taken. The number of samples taken, and the oversampled value are written to the debug stream.
